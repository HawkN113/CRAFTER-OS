# ⚙️ Execution Playbook (CRAFTER)

## Daily

- 1–3 key tasks (Focus)
- Deep work block (Execution)
- Clear outcome defined (Clarity)

---

## Weekly

- Review results
- Optimize processes
- Generate ideas

---

## Rules

- no progress → change approach (Adaptation)
- unfinished tasks → prioritize execution
- overload → restore clarity
